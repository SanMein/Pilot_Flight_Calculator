# Pilot Flight Calculator
A multi-purpose calculator for pilots, flight sim users, and enthusiasts: estimate descent/climb rate, time, horizontal distance, and check if you can safely reach the airport.

## Features
- Calculate vertical speed, time to reach altitude, and horizontal distance
- "Can you reach the airport?" checker for safe approaches
- Quick profile presets for common scenarios and custom profiles
- Supports different units (meters, feet, knots, km/h, mph, km, mi, nmi)
- Modern, responsive UI with instant recalculation
- Multi-language support: English, Russian, Spanish, German
- No installation required — works fully offline in your browser

## How to Use
1. Enter current and target altitudes.
2. Set your speed and desired ADI angle.
3. Choose a profile or adjust parameters manually.
4. Instantly get results for vertical speed, time, and required distance.
5. The calculator will show if you have enough distance for a safe descent or approach.

- All calculations are approximate and for simulation or educational use.
- Real flights require extra factors (wind, aircraft type, weather).

# Pilot Flight Calculator (Русский)
Многофункциональный калькулятор для пилотов, симмеров и энтузиастов: расчёт вертикальной скорости, времени, горизонтальной дистанции и проверки, хватит ли расстояния до аэродрома.

## Возможности
- Расчёт вертикальной скорости, времени достижения высоты и горизонтальной дистанции
- Проверка: хватит ли дистанции до аэродрома для безопасного снижения
- Быстрые профили для типовых сценариев и пользовательские профили
- Поддержка разных единиц (метры, футы, узлы, км/ч, мили/ч, км, мили, морские мили)
- Современный адаптивный интерфейс с мгновенным пересчётом
- Мультиязычность: Английский, Русский, Испанский, Немецкий
- Не требует установки — работает прямо в браузере оффлайн

## Как пользоваться
1. Введите текущую и целевую высоту.
2. Укажите скорость и желаемый угол ADI.
3. Выберите профиль или настройте параметры вручную.
4. Сразу увидите результаты: вертикальная скорость, время и нужная дистанция.
5. Калькулятор покажет, хватает ли дистанции для безопасного снижения или захода.

# Pilot Flight Calculator (Deutsch)
Multifunktionaler Rechner für Piloten, Flugsimulator-Nutzer und Enthusiasten: berechnet Vertikalgeschwindigkeit, Zeit, horizontale Entfernung und prüft, ob die Strecke zum Flughafen ausreicht.

## Funktionen
- Berechnet Vertikalgeschwindigkeit, Zeit zum Erreichen der Höhe und horizontale Entfernung
- Prüft: Reicht die Entfernung für einen sicheren Anflug?
- Schnelle Profile für Standardszenarien und eigene Profile
- Unterstützt verschiedene Einheiten (Meter, Fuß, Knoten, km/h, mph, km, Meilen, Seemeilen)
- Modernes, responsives UI mit sofortiger Neuberechnung
- Mehrsprachig: Englisch, Russisch, Spanisch, Deutsch
- Keine Installation nötig — läuft komplett offline im Browser

## Anwendung
1. Aktuelle und Zielhöhe eingeben.
2. Geschwindigkeit und gewünschten ADI-Winkel einstellen.
3. Profil wählen oder Parameter manuell anpassen.
4. Sie erhalten sofort Ergebnisse für Vertikalgeschwindigkeit, Zeit und erforderliche Entfernung.
5. Der Rechner zeigt, ob die Strecke für einen sicheren Sinkflug reicht.
