# Pilot Flight Calculator

A multi-purpose flight calculation tool for pilots, flight simulation users, and aviation enthusiasts. Version 28 introduces a refined interface and improved performance.

## Key Features
- **Comprehensive Calculations:** Vertical speed, time to reach altitude, and horizontal distance estimation.
- **Safety Checker:** Verify if there's sufficient distance for a safe descent to the airport.
- **Profile Management:** Use quick presets for common scenarios or create custom profiles.
- **Unit Support:** Compatible with various units (meters, feet, knots, km/h, mph, kilometers, miles, nautical miles).
- **Multi-language Support:** Available in English, Russian, Spanish, and German.
- **Modern Interface:** Minimalistic, responsive UI with smooth animations and instant calculations.
- **Offline Access:** Fully functional without installation — runs directly in your browser.

## How to Use
1. Enter the current and target altitudes.
2. Specify your speed and desired ADI angle.
3. Choose a preset profile or adjust parameters manually.
4. View real-time results, including vertical speed, time, and required distance.
5. Check if the distance to the airport is sufficient for a safe descent.

## Notes
- **Positive ADI Angle:** Indicates climb.
- **Negative ADI Angle:** Indicates descent.
- **Approximation:** All calculations are approximate and intended for simulation or educational purposes.
- **Real Flights:** Consider additional factors like wind, aircraft type, and weather.

# Pilot Flight Calculator (Русский)

Многофункциональный инструмент для расчёта параметров полёта для пилотов, пользователей симуляторов и авиационных энтузиастов. Версия 28 приносит обновлённый интерфейс и улучшенную производительность.

## Ключевые возможности
- **Полные расчёты:** Вертикальная скорость, время достижения высоты и горизонтальная дистанция.
- **Проверка безопасности:** Достаточно ли дистанции для безопасного снижения к аэродрому.
- **Управление профилями:** Быстрые пресеты для типовых сценариев и создание собственных профилей.
- **Поддержка единиц измерения:** Метры, футы, узлы, км/ч, мили/ч, километры, мили, морские мили.
- **Мультиязычность:** Доступно на английском, русском, испанском и немецком языках.
- **Современный интерфейс:** Минималистичный, адаптивный UI с плавными анимациями и мгновенными расчётами.
- **Работа оффлайн:** Полнофункционально без установки — работает прямо в браузере.

## Как пользоваться
1. Введите текущую и целевую высоту.
2. Укажите скорость и желаемый угол ADI.
3. Выберите профиль или настройте параметры вручную.
4. Просмотрите результаты в режиме реального времени, включая вертикальную скорость, время и расстояние.
5. Проверьте, хватает ли дистанции до аэродрома для безопасного снижения.

# Pilot Flight Calculator (Deutsch)

Ein vielseitiges Berechnungstool für Piloten, Nutzer von Flugsimulatoren und Luftfahrtbegeisterte. Version 28 bietet eine verfeinerte Oberfläche und verbesserte Leistung.

## Hauptfunktionen
- **Umfassende Berechnungen:** Vertikale Geschwindigkeit, Zeit bis zur Zielhöhe und horizontale Entfernung.
- **Sicherheitsprüfung:** Überprüfe, ob die Entfernung für einen sicheren Sinkflug ausreicht.
- **Profilverwaltung:** Schnelle Voreinstellungen für Standardszenarien oder benutzerdefinierte Profile.
- **Einheitenunterstützung:** Kompatibel mit verschiedenen Einheiten (Meter, Fuß, Knoten, km/h, mph, Kilometer, Meilen, Seemeilen).
- **Mehrsprachige Unterstützung:** Verfügbar in Englisch, Russisch, Spanisch und Deutsch.
- **Moderne Benutzeroberfläche:** Minimalistisches, reaktionsschnelles UI mit flüssigen Animationen und sofortigen Berechnungen.
- **Offline-Zugriff:** Vollständig funktionsfähig ohne Installation — läuft direkt im Browser.

## Anwendung
1. Gib die aktuelle und die Zielhöhe ein.
2. Gib deine Geschwindigkeit und den gewünschten ADI-Winkel an.
3. Wähle ein voreingestelltes Profil oder passe die Parameter manuell an.
4. Sieh dir Ergebnisse in Echtzeit an, einschließlich vertikaler Geschwindigkeit, Zeit und erforderlicher Entfernung.
5. Überprüfe, ob die Entfernung zum Flughafen für einen sicheren Sinkflug ausreicht.
